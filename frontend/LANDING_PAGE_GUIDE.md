## üé® Visual Style Guide

- **Primary:** Blue (#3b82f6)
- **Secondary:** Purple (#8b5cf6)
- **Success:** Green (#10b981)
- **Warning:** Orange (#f59e0b)
- **Danger:** Red (#ef4444)

### Typography
- **Headlines:** Bold, 48-72px
- **Subheadlines:** Regular, 20-24px
- **Body:** 16px
- **Font:** System fonts (similar to Linear/Notion)

### Spacing
- Section padding: 96px vertical
- Card spacing: 32px gap
- Consistent 8px grid system

### Components
- **Cards:** White background, rounded corners, subtle shadow
- **Buttons:** 
  - Primary: Blue, white text
  - Secondary: Gray, dark text
  - Large: 48px height
- **Icons:** Lucide React (modern, minimal)

### Effects
- Hover: Slight shadow increase
- Active: Scale down (0.95)
- Transitions: 200ms ease
